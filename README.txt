Project Title: E-commerce Product Data Manager (File Handling Project)

Contents:
1. Product_Data_Manager.ipynb - The main Jupyter Notebook with full code for loading, updating, and saving data.
2. mainfolder/
   ├── sales_data.csv - Daily sales records for products.
   ├── product_details/ - JSON files containing individual product information.
   └── product_descriptions/ - TXT files with product descriptions.

How to Run:
- Open Product_Data_Manager.ipynb in Jupyter Notebook.
- Follow the prompts to load data, update product info, and save changes.
